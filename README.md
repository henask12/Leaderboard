# 📖 [Leaderboard] <a name="about-project"></a>

A **Leaderboard** List App is a web app that is designed to keep track of and update the results of a game, competition, or activity that involves evaluating the performance of its participants.

<!-- Features -->

### Key Features <a name="key-features"></a>

-   **[Responsive Design]**: The application is designed to be responsive and optimized for different screen sizes and devices.
-   **[Leaderboard Display]**: The application displays a leaderboard with recent scores, showcasing the names and corresponding scores of the top performers.
-   **[Score Submission]**: sers can submit their scores through a form, providing their name and the score achieved.
